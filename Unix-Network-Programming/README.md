- 定义
  - 不是一个实际的协议族，而是一种单个主机的通信方法。
  - 第二章介绍的进程间通信(IPC Inter-Progress Communication) ,实际上就是单个主机上的客户/服务器通信,Unix域协议也可以认为是IPC方法之一。
- 提供两类套接字
  - 字节流套接字(类似TCP)
  - 数据报套接字(类似UDP)
